Bridgiron

ChatGPT ↔ Claude Code Bridge Tool

What is Bridgiron?

Bridgiron streamlines copy-paste operations between ChatGPT and Claude Code. It allows you to copy prompts (instructions for Claude Code) output by ChatGPT to your clipboard with a single click.

Key Features:

  • Automatically extracts prompt sections from ChatGPT responses
  • One-click copy via bookmarklet
  • Reliable prompt identification using SOP/EOP markers

Initial Setup

1
Launch the GUI Tool
Double-click Bridgiron.exe to launch.
2
Set Custom Instructions in ChatGPT
Click "Copy Custom Instructions", open ChatGPT settings, and paste it into the Custom Instructions field.

The location of the settings may vary depending on the ChatGPT version.

3
Register the Bookmarklet
  1. Copy the title using "Copy Title" button
  2. Right-click Chrome's bookmark bar → "Add page..."
  3. Paste the title in the Name field
  4. Return to GUI and copy code using "Copy Code" button
  5. Paste the code in the URL field
  6. Click "Save"
4
Done!
Click the bookmark on the ChatGPT page to copy the prompt to your clipboard.

Button Descriptions

Bookmarklet Section

Copy Title
Copies the title to use for the bookmark name. You can edit it in the input field.
Copy Code
Copies the JavaScript code for the bookmarklet. Configuration file contents (keywords.txt, etc.) are embedded.

ChatGPT Settings Section

Copy Custom Instructions
Copies instructions to make ChatGPT output prompt markers (---SOP---/---EOP---). Set this in ChatGPT's "Custom instructions".

Configuration Files Section

keywords.txt / phrases.txt / delimiters.txt
Opens each configuration file in the default text editor. After editing, generate a new bookmarklet with "Copy Code".
settings.txt
Opens the GUI settings file. Contains language, project path, mini-window position, etc.

Help Section

View Guide
Opens this help page in your browser.

History Section

GPT→CC
Shows history of prompts copied from ChatGPT. Up to 50 entries are saved. Click or use number keys (1-9) to select and copy to clipboard.
CC→GPT
Shows history of Claude Code report copies. Up to 50 entries are saved. Click or use number keys (1-9) to select and copy to clipboard.

Troubleshooting

"Prompt not found" message appears

  • Verify that ChatGPT's custom instructions are set correctly
  • Check if ChatGPT's response contains ---SOP--- and ---EOP---
  • For older conversations without markers, legacy pattern matching is used (yellow notification)

Nothing happens when clicking the bookmark

  • Verify you're on the ChatGPT page (chatgpt.com or chat.openai.com)
  • Check that the bookmark URL starts with javascript:
  • Try copying and registering the code again

Changes to configuration files aren't reflected in bookmarklet

After editing configuration files, you need to generate new code with "Copy Code" and update the bookmark.

About Notification Colors

  • Green: Prompt extracted using SOP/EOP markers (recommended)
  • Yellow: Prompt extracted using legacy pattern matching
  • Red: Failed to extract prompt

Keyboard Shortcuts

  • Alt+C - Copy CC Report
  • 1-9 - Select item in history window
  • Esc - Close history window

File Structure

[Installation Directory]
C:\Program Files (x86)\Bridgiron\
├── Bridgiron.exe         # GUI tool (executable)
├── _Config\              # Initial config files (reference)
├── _DOC\
│   └── Readme.html       # This file
└── images\
    └── ...               # Icons & banner images

[Configuration Files (AppData)]
%APPDATA%\Bridgiron\
├── settings.txt          # GUI settings
├── keywords.txt          # Prompt start keywords
├── phrases.txt           # Prompt start phrases
└── delimiters.txt        # Delimiters

Configuration files are stored in each user's AppData folder.
Type %APPDATA%\Bridgiron in Explorer's address bar to open it.
